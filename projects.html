Swift Swift Trivia App

Summary:

This is the first app I created using Swift and Xcode. For this project, I decided to make a trivia app with questions based on Taylor Swift. As of now, there's 10 questions total, but more questions can be added. There is a welcome screen, n question pages (n being number of questions), and a score screen. The user has the option to retake the quiz too.
